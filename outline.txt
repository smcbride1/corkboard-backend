Cork Board

Virtual cork board that allows users to create and move around virtual notes

MODELS

(Priority) User
	string: name
	string: email
	string: password_digest
	string: facebook_uuid

(Priority) Board
	string: name
	integer: wall_id

(Priority) Wall
	string: name
	integer: room_id

(Priority) Room
	string: name

(Priority) Note
	string: title
	string: short_content
	string: long_content

(Bonus) Connector
	string: target_type
	integer: source_id
	integer: destination_id
	boolean: directional

(Bonus) Theme
	string: font_color
	string: card_color
	string: accent_color
	string: card_shape
